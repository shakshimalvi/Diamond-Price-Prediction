# Diamond-Price-Prediction

An end-to-end Machine Learning project focused on analyzing diamond characteristics and predicting diamond price categories using classification algorithms.

## 🚀 Project Overview

This project uses the Diamonds dataset to classify diamonds into different price categories based on features such as:

* Carat
* Cut
* Color
* Clarity
* Dimensions (x, y, z)
* Depth
* Table

The project covers the complete ML workflow from data preprocessing to deployment using Streamlit.

* Logistic Regression

## 💻 Deployment

The model was deployed using Streamlit to create an interactive web application for real-time predictions.

Run the app locally:

```bash
streamlit run app.py
```

---

## 📌 Key Insights

* Carat has the highest impact on diamond price.
* Better cut, clarity, and color generally increase diamond value.
* Random Forest performed best among all classification models.
